Detailed Description

template<class T, std::size_t N>
class utils::cluster< T, N >

A cluster generated by the k-means algorithm.

Template Parameters
TThe type of the elements in a data vector. Must be an arithmentic type.
NThe dimensional cardinality of the space containing data vectors and clusters.

Instances of this class represent clusters generated by the k-means algorithm.

Applications using k-means should not define specializations of this template or construct instances of any tempate class instantiating this template. The k_means class template will take care of that for you.
