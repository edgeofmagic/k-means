Detailed Description

template<class T, std::size_t N>
class utils::euclidean_vector< T, N >

A vector in the problem space.

Template Parameters
TThe scalar type of vector components. std::is_arithmetic<T>::value must be true.
NThe size of a vector, corresponding to the dimensional cardinality of the problem space. N must be greater than one.

euclidean_vector represents a vector in the problem space (that is, the space within which clusters are partitioned). An instance of euclidean_vector may represent a data point, an abstract point (such as a cluster centroid), or a sum (or difference) of vectors. euclidean_vector derives from std:array, inheriting its public interface. If the dimensional cardinality of a k_means instance is 1, this class will not be used to represent values in the problem space; they will be scalar values.
